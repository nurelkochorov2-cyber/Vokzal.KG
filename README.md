# Vokzal.KG

Vanilla JS приложение для поиска попутчиков и водителей.

## Структура

- `index.html` — главная страница, вкладки и секции.
- `details.html` — детальная страница объявления.
- `style.css` — все стили и адаптивность.
- `app.js` — инициализация и обработчики.
- `storage.js` — LocalStorage и mock-данные.
- `render.js` — функции отрисовки карточек/деталей.

## Запуск

Откройте `index.html` в браузере (или через Live Server).

## Демо-функции

- вкладки `Такси` / `Пассажиры`;
- создание объявления с выбором роли в момент публикации;
- профиль пользователя;
- кликабельные кнопки связи (`wa.me`, `t.me`, `tel:`);
- счетчик мест и переключатель `Мест нет` для своих водительских объявлений;
- секция `Мои объявления` (редактирование статуса/удаление);
- переход на `details.html?id=...`.
